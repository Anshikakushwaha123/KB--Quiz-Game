🎮KBC Quiz Game

An interactive quiz game inspired by Kaun Banega Crorepati (KBC).
Users can login, play quiz rounds, and test their knowledge.

📌 Features

🔐 User Login & Signup
❓ Multiple Choice Questions
🧠 Score Tracking
🎯 Interactive UI
⚡ Fast and responsive design

🛠️ Tech Stack

* HTML
* CSS
* JavaScript

📂 Project Structure

* index.html → Dashboard / Home Page
* login.html → Login Page
* signup.html → Create Account
* game.html → Quiz Game
* result.html → Score Page
